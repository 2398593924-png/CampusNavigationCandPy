# 基于迪杰斯特拉算法的校园导航系统

本项目使用C语言和Python进行编写，通过迪杰斯特拉算法实现了可自定义节点和路线的校园导航系统，以及路线定义及最短路径的可视化。为满足实际生活需要，本项目还额外提供了简单的用户界面以便于用户自定义新地点。

值得注意的是，本项目使用json数据包实现了C语言和Python之间的数据传输，其中C语言部分采用了开源的cJSON库（仓库链接：DaveGamble/cJSON: Ultralightweight JSON parser in ANSI C）实现了json文件的读取和解析，Python端则使用OpenCV库实现可视化操作。最短路径的获取采用了简单的迪杰斯特拉算法，便于演示。

## 快速开始

本项目的文件列表如下：

| 文件名 | 介绍 |
| :--- | :--- |
| cJSON.c / cJSON.h | cJSON库文件 |
| Dijkstra.h | 包含Dijkstra算法实现的头文件 |
| GetAdjMatrix.h | 读取并解析json用于构建邻接矩阵 |
| main.cpp | 主程序 |
| map.jpg | 地图图像 |
| ShowPath.py | 用于可视化最短路径 |
| tool.py | 用于自定义节点和边 |
| temp.bin | 数据通信所产生的临时文件 |
| graph_data.json | 保存的图结构文件 |

首先，必须确保地图文件放置于程序根目录下，并命名为map.jpg
随后使用以下命令启动地图设置程序：py tool.py
  

## 食用方法：
本程序提供了图形化的按钮，也可以使用键盘进行操作，具体如下：

- 程序默认处于节点模式。点击“Add Node“按钮或N键可以切换到此模式，接下来使用鼠标左键点击你想设置的每个地点，程序将自动标号并显示在图片上。
- 当节点设置完成后，可以点击“Add Edge”按钮或E键切换到边模式。正确进入边模式后，首先点击起始节点，程序将在终端给出提示，随后不断沿着路径点击，直到到达终末节点，程序将自动显示您绘制的路线，如右图所示。
- 如果对当前的节点或路线设置不满意，可以点击“Clear All”按钮或C键清空所有设置。如果设置完成，则点击“Save”或S键进行数据的保存，随后点击键盘Q键即可退出程序。如果一切正常，你将在根目录看到程序创建的graph_data.json文件。
- 接下来编译并运行main.cpp（注意各文件是否正确放置在根目录下，否则程序会报错），程序会根据上述步骤创建的json文件给出各节点对照表
 
根据对照表及提示输入你的起始点（这里需要输入对照表左端的数字）及目的地，按下回车程序将自动给出最短路径的像素距离及具体的路径，并通过调用python程序自动给出最短路径的可视化，至此程序结束。
